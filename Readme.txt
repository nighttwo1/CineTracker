// 개인 과제로 진행하였습니다.


프로젝트 설명:
캘린더의 일정을 참고하여 영화를 예매하는 앱으로 구글 캘린더에 저장된 일정의 시간을 확인하여 가능한 시간대의 상영시간표 목록을 불러올 수 있도록 하였습니다.

프로젝트에 넣은 기능:
구글 API를 활용하여 로그인과 구글 캘린더 정보를 불러옵니다.
웹 parsing을 이용하여 네이버 현재 상영정보, CGV의 상영시간표를 불러옵니다.

프로젝트 내용:
최대한 현실의 예매 기능을 반영하기 위해서 현재의 날짜 이후로부터 약 1주(영화 예매가 가능한 시기)간의 정보를 불러올 수 있도록 반영하였으며 예매된 영화시간이 지나면 자동으로 삭제될 수 있도록 만들었습니다.

아쉬웠던 점:
1. website parsing 기능을 넣다보니 웹사이트가 점검중이거나 기타 등등의 문제가 발생하게 되었을 경우 정보를 제대로 받아올수 없었습니다.
2. 영화에 대한 정보를 네이버의 현재 상영정보에서 불러오다보니 개봉 예정의 영화들은 선택할 수 없게 되어 실제 영화 예매 방법과 차이가 생기게 되었습니다.
